Agent Search Help
Search Criteria State or First Name must be combined with 1 or more search value(s) to produce a valid search. All other searches may be performed with a single criteria value.
License Number Enter the exact number. A license number is a unique state 6 digit or longer number.
National Producer Number (NPN) Enter the exact NPN number.
Zip Code Enter the 5 digit US Zip Code.
Type of Insurance Only one type of insurance may be selected per search. May not use this criteria in combination with 'License Type'.
License Type Only one license type may be selected per search. May not use this criteria in combination with 'Type of Insurance'.

Company Search Help
Single Value Searches Enter Company Number or Company Name or Company Type or Line of Authority or any combination.
Company Type Defines the specific industry in the insurance market. For example, property and casualty.
Line of Authority Type of insurance a company is licensed to sell. Note some companies may be authorized to sell a type of insurance, but is not actively selling it. Contact the company, or an agent or agency that represents the company and ask if the company is selling a particular type of insurance.

Navigator Search Help
Simple Search Use the radio buttons to designate Entity or Individual then press Search button to display a list of all Navigators of that type registered in Virginia.
Single Value Search Enter any one of the listed fields, or any combination, to search for a specific match; however, you must designate whether you are searching for an Entity or Individual.
Sort Results Select a value in the Sort By dropdown to designate how the results listing is presented for viewing -- Name, City or State.
